# Command Line Help

> tensai-suraimu.exe --help

```
tensai-suraimu 0.1.0
zhengxiaoyao0716

Tensai Suraimu to Baka Meido
offical site: https://github.com/tensai-suraimu

USAGE:
    tensai-suraimu.exe [OPTIONS]

OPTIONS:
    -h, --help
            Print help information

    -l, --language <language>
            for example: 简体中文, 日本語, ...

    -s, --skip
            skip initial page

    -T, --theme-mode <theme-mode>
            [possible values: light, dark]

    -V, --version
            Print version information

    -w, --window-size <window-size>
            format: {width}*{height}

    -W, --window-mode <window-mode>
            [possible values: window, frameless, fullscreen]
```
